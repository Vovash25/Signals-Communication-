CDMA Simulation Project
Цей проєкт моделює базовий процес передачі даних за технологією CDMA (Code Division Multiple Access). Скрипт зчитує вхідні повідомлення та унікальні коди для кожного клієнта, проводить кодування, створює сумарний сигнал та демонструє процес відновлення (декодування) початкових даних.
