Digital Modulation Analysis Project
Цей проєкт фокусується на дослідженні та порівнянні характеристик різних методів цифрової модуляції (ASK, FSK, BPSK, QPSK, QAM-8 та QAM-16). Основна увага приділяється аналізу часових діаграм, спектральної щільності потужності (PSD) та ефективності використання смуги частот.
